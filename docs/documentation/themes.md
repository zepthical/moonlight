## Load own theme
```lua
WindUI:AddTheme({
    Name = "Halloween",
    
    Accent = "#331400",
    Outline = "#400000",
    
    Text = "#EAEAEA",
    PlaceholderText = "#AAAAAA"
})
```

## Use your theme
```lua
WindUI:SetTheme("Halloween")
```
